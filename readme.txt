== Description ==

OS Materializecss PopUp is the colorful, responsive, customizable popup plugin for WordPress using materialize css.

== Features ==
* Admin interface to add, edit and manage materializecss popup
* Different materializecss popup types
* Design customization options for popup
* Enhanced display features

Shortcode

We have created four types of display methods

[osmp-materializecss-popup id="{post_id}" design="{design-type}"]

Support

So that others can share in the answer, please submit your support requests through the WordPress forums for OS Materializecss PopUp(http://wordpress.org/support/plugin/os-materializecss-popup).

== Installation ==

Minimum requirements:    
    
 Wordpress 4.3+    
 PHP 5.4x    
 MySQL 5.2x+  

Installation from within backend:

1. In plugin manager, click "Add New".
2. Search for "os-materializecss-popup".
3. Click "install".
4. Activate the plugin once it is installed.
5. Click on "OS Materializecss PopUp" from the left menu and follow the instructions to complete the installation.

FTP upload installation method:

1. Upload the `os-materializecss-popup` folder to the `/wp-content/plugins/` directory using your FTP client.
2. Activate the plugin through the 'Plugins' menu.
3. Go to the OS Materializecss PopUp menu, finish the installation and create a materializecss popup.

Zip upload installation method (make sure uploads up to 5MB are allowed for your hosting):

1. Login to your WordPress site administrator panel and head over the 'Plugins' menu.  
2. Click 'Add New'.
3. Choose the 'Upload' option.
4. Click **Choose file** (**Browse**) and select the os-materializecss-popup.*.zip file.   
5. Click **Install Now** button.    
6. Once it is complete, activate the plugin.   
7. Go to the OS Materializecss PopUp menu, finish the installation and create a materializecss popup.
